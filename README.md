# Medical Appointment Booking

A React-based healthcare platform where users can register, login, search doctors, book appointments, and give reviews.

---

## 🚀 Features

- User Registration (Sign Up)
- User Login
- Search Doctors
- Book Appointment
- Cancel Appointment
- Give Reviews
- Logout Functionality
- Notification System

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- HTML
- CSS

---

## 📂 Folder Structure

src/
  components/
  pages/
  App.js
  index.js

---

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   npm install
3. Run the app:
   npm start
